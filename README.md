# Getting Started
## Prerequisites
- Python 3.8
- [poetry 1.0.5](https://python-poetry.org/docs/)

## Setup Environment
```
$ poetry install
$ export PYTHONPATH=$PYTHONPATH:$PWD
```

## Run Program
```
$ poetry run python problem1/problem1.py
```

## Run Unit Test
```
$ poetry run pytest
```
